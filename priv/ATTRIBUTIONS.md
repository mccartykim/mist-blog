### Assets Used

From the Hugo Bear Cub theme, we extracted and adapted:
- CSS styling (`original.css` and `syntax.css`)
- HTML template structure and layout concepts
- Color scheme and typography choices

The CSS and HTML templates have been reimplemented in pure Nix for this static site generator.

## Hugo

This site was originally built with [Hugo](https://gohugo.io/), a fast and flexible static site generator written in Go.

Thank you to the Hugo team and the Bear Cub theme authors for their excellent work that inspired this pure Nix implementation!
